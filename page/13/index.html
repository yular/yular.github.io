Hackerrank Data Structure Problem Solution List III

Here is the summary of Hackerrank data structure problem solution list.
summary list

Below is the detailed explanation of the idea behind solution.

Maximum Element

Problem Link
Code Link
Data structure problem. Use two stacks. One is to save all the given elements that are not poped out and the other is to save the maximum value starting from the element at the bottom of the first stack.

Largest Rectangle

Problem Link
Code Link
Data structure problem. Add 0 to the end of given array. Starting from the head of array, push the index of every element into a stack. If the element value pointed by the index at the top of stack is lower than that of the current index, pop out that index and then calculate the area formed between this index and the new top index at the stack; and update the answer if it is smaller than the new area.

Simple Text Editor

Problem Link
Code Link
Data structure problem. Use an array to simulate a stack.

Poisonous Plants

Problem Link
Code Link
Data structure problem. Use two stacks to do the simulation for the problem.

Waiter

Problem Link
Code Link
Data structure problem. At first find all the prime needed. Then use stack to do the simulation for the problem.

Jesse and Cookies

Problem Link
Code Link
Basic problem. Use queue to simulate the problem.

Castle on the Grid

Problem Link
Code Link
BFS problem. Find the path from source to destination with least number of changing direction.

Down To Zero II

Problem Link
Code Link
Basic problem. Use BFS to simulate the process.

Truck Tour

Problem Link
Code Link
BFS problem. Use BFS to simulate the process.

Queries with Fixed Length

Problem Link
Code Link
Monotone problem. Elements inside the queue is in non-increasing order. When insert element into the queue, starting from the tail of queue, pop back elements that has smaller value than the current one; and then starting from the head of queue, pop out the elements that are out of required boundary. Now in every query, the value of element at the head of queue is answer.

QHEAP1

Problem Link
Code Link
Priority queue problem. Use two min priority queues, one is to save all the values that have been removed and the other is to save all given values. Before output the minimum value, pop out the top value of the two priority queues until their top values are not equal.

Find Median

Problem Link
Code Link
Priority queue problem. Use one max priority queue and one min priority queue. Both of them save half of the given elements. The rule is that given a new element, if its value is larger than the value of top element of max priority queue, push that element into min priority queue; otherwise, push it into max priority queue. Now if difference between the the total number of elements of these two queues are larger than one, rebalance the queues so that the difference is no larger than one. For each query, if the total number of elements of the two queues are odd, the answer is the value of top element of max priority queue; otherwise, it is the half of the sum of the top value of two priority queues.

Minimum Average Waiting Time

Problem Link
Code Link
Greedy problem. Sort the order based on this rule: if two orders have the same start time, the one cost less time come first; otherwise, the order with less time come first. Then use priority queue to go through the sorted orders one by one and each time put the remaining order that cost most time to the top of priority queue.

Hackerrank Data Structure Problem Solution List II

summary list

Below is the detailed explanation of the idea behind solution.

Tree: Preorder Traversal

Problem Link
Code Link
Basic problem. Recursively visit the root, left child and right child of every subtree.

Tree: Postorder Traversal

Problem Link
Code Link
Basic problem. Recursively visit the left child, right child, and root of every subtree.

Tree: Inorder Traversal

Problem Link
Code Link
Basic problem. Recursively visit the left child, root, and right child of every subtree.

Tree: Height of a binary tree

Problem Link
[Code Link](https://github.com/yular/CC--InterviewProblem/blob/master/Hackerrank_InterviewStreet/tree-height-of-a-binary-tree.cpp
Basic problem. Use DFS to find the longest path from root to leaves.

Tree : Top View

Problem Link
Code Link
Basic problem. Use dfs to find the path from root to left most leaf and right most leaf; and then use bfs to output the result.

Tree: Level Order Traversal

Problem Link
Code Link
Basic problem. Use bfs to do the level search.

Binary Search Tree : Insertion

Problem Link
Code Link
Basic problem. Use dfs to find the position for insertion and then add the give node there.

Tree: Huffman Decoding

Problem Link
Code Link
Greedy solution. Use priority queue. Every time get the two nodes with least frequency and create a new node whose frequency is the sum of them, and set it as the root of its children, put it back to priority queue. This process continues until there is only one node in the queue.

Binary Search Tree : Lowest Common Ancestor

Problem Link
Code Link
Basic problem. Use dfs to find the two targeted nodes while at the same time save the parent node of those nodes in the path. And then find the parent of nodes recursively until a common parent node is found.

Swap Nodes [Algo]

Problem Link
Code Link.
Basic problem. Use level search.

Self Balancing Tree

Problem Link
Code Link
Data structure problem. Find the position to insert the node using binary search similar approach; after insert the node, rotate the subtree if needed.

Median Updates

Problem Link
Code Link
Data structure problem. Use Sized Balanced Tree and find the n/2 largest elements among all the nodes where n is the total number of nodes of tree currently.

Hackerrank Data Structure Problem Solution List I

Here is the summary of Hackerrank data structure problem solution list.
summary list

Below is the detailed explanation of the idea behind solution.

Arrays- DS

Problem Link
Code Link
Basic problem. Read and save all the integers into an array and then output them in reversal order.

2D Array - DS

Problem Link
Code Link
Basic problem. Enumerate upper left corner of each hour glass and compute its sum, and then find the largest one.

Dynamic Array

Problem Link
Code Link
Basic problem. Based on the operation type (1 or 2), do the operation suggested in the problem statement. It should be very straightforward.

Sparse Array

Problem Link
Code Link
Basic problem. Use Hash Map to save the number of times given strings appear in the list, and then output that value if the query string is found, otherwise, output 0 instead.

Print the Elements of a Linked List

Problem Link
Code Link
Basic problem. Use a pointer to go through every element from head to tail and print the value of them.

Insert a Node at the Tail of a Linked List

Problem Link
Code Link
Basic problem. At first reach the end of given linked list and then insert the given element there; if the original linked list is empty, just use the given element as head of a new linked list.

Insert a node at the head of a linked list

Problem Link
Code Link
Basic problem. Just use the given element as the new head of given linked list.

Insert a node at a specific position in a linked list

Problem Link
Code Link
Basic problem. Just move the pointer to the prior position of given one, and then insert the given element between them.

Delete a Node

Problem Link
Code Link
Basic problem. Just move the pointer to the prior element of given one, and then remove the next element.

Print in Reverse

Problem Link
Code Link
Basic problem. Just visit every element from head to tail and save the value into an array; and then print the value of array elements in reverse order.

Reverse a Linked List

Problem Link
Code Link
Basic problem. Just use two pointers and swap every two adjacent elements from head to tail.

Compare two Linked List

Problem Link
Code Link
Basic problem. Use two pointers and each of them goes through the linked list and compare the element one by one.

Merge two sorted Linked List

Problem Link
Code Link
Basic problem. Use two pointers and each of them goes through the linked list and compare the element one by one. In every comparison, the smaller element will be added to final linked list.

Get Node Value

Problem Link
Code Link
Basic problem. Use a pointer to find the total length of linked list and then caculate the position that we can get the final value.

Delete duplicate-value nodes from a sorted linked list

Problem Link
Code Link
Basic problem. Use two pointers to go through the given linked list from head to tail. One points to the next element of the other. If the value of them are equal, the delete the node closed to tail.

Detect a Cycle

Problem Link
Code Link
Basic problem. Use two pointers. One goes through the linked list one step each time and the other goes through the linked list two steps each time. If they will meet at one node, then there is a cycle.

Find Merge Point of Two Lists

Problem Link
Code Link
Basic problem. Set the next element of tail as the one of the heads.

Insert a node into a sorted doubly linked list

Problem Link
Code Link
Basic problem. Just use one pointer to figure out the insertion position and then add the new node there.

Reverse a doubly linked list

Problem Link
Code Link
Basic problem. Use the similar approach of reversing a single linked list.
