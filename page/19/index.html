Wiggle Sort

A classic Google Interview problem. Start from index one, and then check whether is the largest one among its two adjacent numbers; if not, swap it with the largest one; after that go to the next two element until reaching the end.
经典的谷歌面试题。从第二个元素开始，比较当前元素和与其相邻两个元素的大小，将最大那个与它交换位置；然后每次移动到下两个元素，直到到达数组末尾。

###Add Digits(Easy)

###Ugly Number(Easy)
Continue to divide the given number by 2, 3 and 5 before it equals to one or it cannot be divided by all these three numbers evenly. If the number is one, return true; otherwise, return false.
将给定的数字不断除以2，3和5直到不能整除它们为止。此时若结果为1则返回true，否则false。
Read Code

###Ugly Number II (Medium)
Use Min Heap and Hash Set. Insert integer 1 into heap and count how many numbers have been poped out of the heap. While the count is less than the threshold, every time we pop out the number of top of heap and then multiply it by 2, 3 and 5; we only push the new number into heap if it does not appear before and it should be no larger than INT_MAX. The last number that we get before break out from the while loop is final answer.
用小顶堆和哈希表。初始化时将1插入堆中，然后开始计数。在计数器一直不大于给定数时，每次迭代取出堆顶数，将该数字分别乘以2，3和5，若结果没有溢出，就插入堆中。最后一次迭代的结果就是答案。
Read Code

###3 Sum Smaller (Medium)
Just two loops. At first, sort the array. Then, in the outer loop, go through each integer from end to the third position; in the inner loop, use two pointers to figure out the range that the sum of every two combinations plusing the one from outer loop still less than target integer.
先排序。然后两层循环。外层循环从末尾开始枚举到第三个位置；内层循环则用头尾两个指针，一个从数组第一个元素开始，另一个从外循环当前指向的元素的前一个元素开始，一起相向枚举，找出合要求的数组元素组合范围，并将符合要求的组合数目相加即可。
Read Code

Linux Terminal Shortcut

Here is the link of the list of Linux Shortcut keys: Shortcut

The list is also here:

  • Ctrl + A: Moves the cursor to the start of a line.
  • Ctrl + E: Moves the cursor to the end of a line.
  • Esc + B: Moves to the beginning of the previous or current word.
  • Ctrl + K: Deletes from the current cursor position to the end of the line.
  • Ctrl + U: Deletes from the start of the line to the current cursor position.
  • Ctrl + W: Deletes the word before the cursor.
  • Alt + B: Goes back one word at a time.
  • Alt + F: Moves forward one word at a time.
  • Alt + C: Capitalizes letter where cursor is and moves to end of word.
