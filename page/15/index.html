Maven Including All Dependencies into One Jar Tutorial

When we compile Java Maven project into a jar file, we should include third party jars that are used in the program; otherwise, our programs will fail because of No Class Found Error while running. Here are some ways to include all third party indenpencies into only one jar.

1.Just download the third party jars, save them into project directories and then compile and create our program jar file using command line.

2.Use Maven Assemble Plugin in our project. In this case, we will get only one jar for our project and all dependencies are included into that jar file. Here is the detailed tutorial.

Excel: Frequently Used Excel Function Summary

Here is the list of Excel functions that may be used frequently in data analysis.

SUM

Caculate the sum of data in a specific area.
Ex: =SUM(A1:A5)

AVERAGE

Caculate the average of data in a specific area.
Ex: =AVERAGE(A1:A5)

MAX

Find the maximum value of data in a specific area.
Ex: =MAX(A1:A5)

MIN

Find the minimum value of data in a specific area.
Ex: =MIN(A1:A5)
