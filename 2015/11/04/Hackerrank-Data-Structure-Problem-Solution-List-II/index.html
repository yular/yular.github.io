Hackerrank Data Structure Problem Solution List II

Here is the summary of Hackerrank data structure problem solution list.
summary list

Below is the detailed explanation of the idea behind solution.

Tree: Preorder Traversal

Problem Link
Code Link
Basic problem. Recursively visit the root, left child and right child of every subtree.

Tree: Postorder Traversal

Problem Link
Code Link
Basic problem. Recursively visit the left child, right child, and root of every subtree.

Tree: Inorder Traversal

Problem Link
Code Link
Basic problem. Recursively visit the left child, root, and right child of every subtree.

Tree: Height of a binary tree

Problem Link
[Code Link](https://github.com/yular/CC--InterviewProblem/blob/master/Hackerrank_InterviewStreet/tree-height-of-a-binary-tree.cpp
Basic problem. Use DFS to find the longest path from root to leaves.

Tree : Top View

Problem Link
Code Link
Basic problem. Use dfs to find the path from root to left most leaf and right most leaf; and then use bfs to output the result.

Tree: Level Order Traversal

Problem Link
Code Link
Basic problem. Use bfs to do the level search.

Binary Search Tree : Insertion

Problem Link
Code Link
Basic problem. Use dfs to find the position for insertion and then add the give node there.

Tree: Huffman Decoding

Problem Link
Code Link
Greedy solution. Use priority queue. Every time get the two nodes with least frequency and create a new node whose frequency is the sum of them, and set it as the root of its children, put it back to priority queue. This process continues until there is only one node in the queue.

Binary Search Tree : Lowest Common Ancestor

Problem Link
Code Link
Basic problem. Use dfs to find the two targeted nodes while at the same time save the parent node of those nodes in the path. And then find the parent of nodes recursively until a common parent node is found.

Swap Nodes [Algo]

Problem Link
Code Link.
Basic problem. Use level search.

Self Balancing Tree

Problem Link
Code Link
Data structure problem. Find the position to insert the node using binary search similar approach; after insert the node, rotate the subtree if needed.

Median Updates

Problem Link
Code Link
Data structure problem. Use Sized Balanced Tree and find the n/2 largest elements among all the nodes where n is the total number of nodes of tree currently.
