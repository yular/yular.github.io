Hackerrank Data Structure Problem Solution List I

Here is the summary of Hackerrank data structure problem solution list.
summary list

Below is the detailed explanation of the idea behind solution.

Arrays- DS

Problem Link
Code Link
Basic problem. Read and save all the integers into an array and then output them in reversal order.

2D Array - DS

Problem Link
Code Link
Basic problem. Enumerate upper left corner of each hour glass and compute its sum, and then find the largest one.

Dynamic Array

Problem Link
Code Link
Basic problem. Based on the operation type (1 or 2), do the operation suggested in the problem statement. It should be very straightforward.

Sparse Array

Problem Link
Code Link
Basic problem. Use Hash Map to save the number of times given strings appear in the list, and then output that value if the query string is found, otherwise, output 0 instead.

Print the Elements of a Linked List

Problem Link
Code Link
Basic problem. Use a pointer to go through every element from head to tail and print the value of them.

Insert a Node at the Tail of a Linked List

Problem Link
Code Link
Basic problem. At first reach the end of given linked list and then insert the given element there; if the original linked list is empty, just use the given element as head of a new linked list.

Insert a node at the head of a linked list

Problem Link
Code Link
Basic problem. Just use the given element as the new head of given linked list.

Insert a node at a specific position in a linked list

Problem Link
Code Link
Basic problem. Just move the pointer to the prior position of given one, and then insert the given element between them.

Delete a Node

Problem Link
Code Link
Basic problem. Just move the pointer to the prior element of given one, and then remove the next element.

Print in Reverse

Problem Link
Code Link
Basic problem. Just visit every element from head to tail and save the value into an array; and then print the value of array elements in reverse order.

Reverse a Linked List

Problem Link
Code Link
Basic problem. Just use two pointers and swap every two adjacent elements from head to tail.

Compare two Linked List

Problem Link
Code Link
Basic problem. Use two pointers and each of them goes through the linked list and compare the element one by one.

Merge two sorted Linked List

Problem Link
Code Link
Basic problem. Use two pointers and each of them goes through the linked list and compare the element one by one. In every comparison, the smaller element will be added to final linked list.

Get Node Value

Problem Link
Code Link
Basic problem. Use a pointer to find the total length of linked list and then caculate the position that we can get the final value.

Delete duplicate-value nodes from a sorted linked list

Problem Link
Code Link
Basic problem. Use two pointers to go through the given linked list from head to tail. One points to the next element of the other. If the value of them are equal, the delete the node closed to tail.

Detect a Cycle

Problem Link
Code Link
Basic problem. Use two pointers. One goes through the linked list one step each time and the other goes through the linked list two steps each time. If they will meet at one node, then there is a cycle.

Find Merge Point of Two Lists

Problem Link
Code Link
Basic problem. Set the next element of tail as the one of the heads.

Insert a node into a sorted doubly linked list

Problem Link
Code Link
Basic problem. Just use one pointer to figure out the insertion position and then add the new node there.

Reverse a doubly linked list

Problem Link
Code Link
Basic problem. Use the similar approach of reversing a single linked list.
