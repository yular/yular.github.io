npm commands summary

Here is short summary of some basic npm commands:

npm init

Create a Node JS project directory with some basic files and directories such as node_modules and package.json.

npm ls

List local or global (with -g option) package installed.

npm start

Start a Node JS server of the specific project. This runs an arbitrary command specified in the package “start” property of its “scripts” object. If no “start” property is specified on the “scripts” object, it will run node server.js.
