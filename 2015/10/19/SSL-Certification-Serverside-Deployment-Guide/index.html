Here is the simple guide to deploy SSL certification on server side. Currently, we use Tomcat8 in this tutorial.

Decrypt the PFX File

In this step, we run the following command to decrypt the pfx file created in the last step.

We will be asked to provide the password for the PFX file in this step. Once we enter that, we will again be asked to provide a new password which is used to encrypt the private key. Please remember this password also. Here we set the new password as “changeit” in our example.

When this step is complete, we should have a PEM file. Open the file in a text editor and copy the private key and certificate to different files. Remember to keep the dashed lines intact when we copy the certificates. There is some additional text above the key, and also between the key and certificate – this text should be ignored and should not be included in the certificate and key files.
