Jupyter Configuration

There are mainly two kinds of Jupyter configuration system. One is common Jupyter configuration system, which is shared by all Jupyter applications. The other is kernel configuration system, which is identified by different kernels.
For common Jupyter configuration system, the conf files are saved under ~/.jupyter by default.
For kernel configuration system, the conf files are saved in different places. For example, the IPython kernel conf files are saved under ~/.ipython and we can find a folder named kernel there.
