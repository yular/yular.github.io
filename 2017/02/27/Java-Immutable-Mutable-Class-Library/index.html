Java Immutable Class QuickStart

Here is the official website of Java Immutable Class, benefits of Immutable Class and Getstarted Tutorial.

Generally, the benefits of Immutable class is:

  • Create thread-safe objects.
  • Easy to make program parallel.
  • Keep the internal program state consistent.
  • Cache the references to immutable objects.

Now below it is our quickstart tutorial.

Setup Environment

Here we suggest to use IntelliJ IDE because this IDE is easy to generate immutable class.

Here is the official tutorial to enable annotations for immutable class feature.

Create Java Project

In IntelliJ, create a Java Maven Project. This part will be provided later as it should be very easy.

Enable Annotations

Open IntelliJ system preferences panel and edit the settings as follows:
