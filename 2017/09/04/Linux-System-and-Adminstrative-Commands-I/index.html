Users and Groups

users

  • Show all logged on users.

groups

  • List the current user and the gourps s/he belongs to.

chown, chgrp

useradd, userdel

usermod

  • Modify a useraccount’s password, group membership, expiration date and etc.

groupmod

id

  • Lists the real and effective user IDs and the group IDs of the user associated with the current process.

lid

  • Shows the group(s) a given user belongs to, or the users belonging to a given group.

who

w

logname

  • Show current user’s login name.

su

sudo

passwd

  • Sets, changes, or manages a user’s password.

ac

  • Show users’ logged in time.

last

  • List last logged in users.

mewgrp

Terminals

tty

  • Echoes the terminal filename of current users’ terminal.

stty

  • Shows and/or changes terminal settings.

stterm

  • Set certain terminal attributes.

tset

  • Show or initialize terminal settings.

setserial

getty, agetty

mesg

wall

  • Sending a message to all users at every terminal logged into the network.
