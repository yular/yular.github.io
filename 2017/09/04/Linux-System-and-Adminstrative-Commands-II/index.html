Information and Statistics

uname

arch

  • Show system architecture.

lastcomm

  • Gives information of previous commands, as stored in the /var/account/pacct file.

lastlog

lsof

  • List open files.

strace

  • System trace: diagnostic and debugging tool for tracing system calls and signals.

ltrace

  • Library trace: diagnostic and debugging tool that traces library calls invoked by a given command.

nc

  • A complete toolkit for connecting to and listening to TCP and UDP ports.

free

  • Show memory and cache usage in tabular form.

proinfo

  • Extract and list information and statistics from the /proc pseudo-filesystem.

lsdev

  • List devices (installed hardware).

du

  • Show disk file usage recusively.

df

  • Show filesystem usage in tabular form.

dmesg

stat

  • Gives detailed and verbose statistics on a given file or set of files.

vmstat

  • Show virtual memory statistics.

uptime

  • Show how long the system has been running.

hostname

  • Lists the hostname of system.

hostid

  • Echo a 32-bit hexadecimal numerical identifier for the host machine.

sar

readelf

size

  • Show the segment sizes of a binary executable or archive file.

System Logs

logger

  • Append a user-generated message to the system log /var/log/messages.

logrotate

  • Manage the system log files, rotating, compressing, deleting, and/or e-mailing them, as appropriate.
