Network

nmap

  • Network mapper and port scanner.

ifconfig

  • Network interface configuration and tuning utility.

netstat

  • Show current network statistics and information, such as routing tables and active connection.

iwconfig

  • Command set of wireless network configuration.

ip

  • Utility for setting up, changing, and analyzing IP networks and attached devices.

route

  • Show info about or making changes to the kernel routing table.

iptables

chkconfig

  • Check network and system configuration.

tcpdump

  • A tool for analyzing and troubleshooting traffic on a network by dumping packet headers matching specified criteria.

Filesystem

mount

  • Mount a filesystem, usually on an external device.

sync

  • Force an immediate write of all updated data from buffers to hard drive.

losetup

  • Set up and configure loopback devices.
